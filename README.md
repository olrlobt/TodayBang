# TodayBang
<hr> 
부동산 매매와 가구 쇼핑이 한 번에 가능한 웹 쇼핑몰

오늘의 집 (가구 쇼핑몰) + 직방 ( 부동산 매매 사이트)을 클론 코딩하며, Spring과 Java 등의 기술에 익숙해 지는 것을 목표로 한다.

<br><br>
### 이승헌 구현

<hr> 
• 로그인 / 마이페이지 / 관리자 페이지 구현

• 게시판 CRUD 구현
